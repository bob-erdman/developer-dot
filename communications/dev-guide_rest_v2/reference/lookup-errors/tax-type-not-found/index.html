Tax type {taxType} not found

  • Previous

Error Description

The Tax Type passed as part of the /api/v2/afc/taxtype/{taxType} Lookups endpoint is not a known valid tax type.

Example

How to Resolve

Ensure that a valid and numeric tax type is being passed in the {taxType} parameter of the /api/v2/afc/taxtype/{taxType} Lookups endpoint.
